## Abstract
AbstractThis repository contains the data processing and visualization workflow for Angle-Resolved Photoemission Spectroscopy (ARPES) measurements of Bismuth Selenide ($\text{Bi}_2\text{Se}_3$).The included dataset (BiSe_232deg_38p7nm_static_6.h5) captures static photoelectron emission intensities mapped across energy and angle dimensions using a SPECS Phoibos electron analyzer. Excited by $38.7\text{ nm}$ extreme ultraviolet (EUV) radiation ($\approx 32.04\text{ eV}$ photon energy), the spectra record electronic band dispersions at a central kinetic energy of $28.0\text{ eV}$ with a $30.0\text{ eV}$ pass energy.The provided Python scripts and Jupyter Notebooks unpack the raw HDF5 dataset, calibrate the energy and angular axes ($\theta \approx 24.5^\circ \text{ to } 72.1^\circ$), and plot clear 2D momentum maps for analyzing topological surface states.
